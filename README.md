
# 2dpixels-canvas

render 2dpixels to a canvas element

## how

### install

```
git clone https://github.com/ahdinosaur/2dpixels-canvas
cd 2dpixels-canvas
npm install
```

## develop

```
npm start
```

browse to <http://localhost:9966/>.

## test

```
npm test
```

## deploy

```
npm run deploy
```
